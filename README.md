# zh-portal-notes

## 项目简介

zh-portal-notes 是一个用于归档和发布独立 HTML 页面的仓库。  
这里存放的页面多为笔记、参考或工具类内容，不针对任何特定域名或网站。  
每个页面均为自包含的 HTML 文件，可直接在浏览器中打开使用。

## 目录说明

```
zh-portal-notes/
├── README.md          # 本文件
├── index.html         # 可选的导航页面
└── pages/             # 存放独立 HTML 页面的目录
    ├── note-001.html
    ├── note-002.html
    └── ...
```

- `pages/` 目录下为归档的 HTML 页面，文件命名以 `note-` 开头并附加编号。
- 每个页面均不依赖外部资源，可离线使用。
- 如有需要，`index.html` 可作为页面的简单索引列表。

## 页面归档说明

- 所有页面均为独立 HTML 文件，不含外部链接、脚本或样式引用（除标准 HTML 元素外）。
- 页面内容以文字笔记为主，可能包含少量演示性交互（如 JavaScript 示例）。
- 归档不涉及任何具体网站、域名或商业服务。
- 页面标题和内容均保持中立、客观，仅用于学习与参考目的。

## 维护说明

- 仓库由个人维护，不定期更新。
- 欢迎通过 Issues 提交建议或报告问题。
- 不接受涉及具体域名、网站或推广性质的提交。
- 所有页面内容应遵守开源社区规范，不包含违法或侵权信息。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/zh-portal-notes.git
   ```
2. 直接打开 `pages/` 目录下的 HTML 文件即可查看。
3. 也可通过 GitHub Pages 等静态托管服务部署（需自行配置）。
